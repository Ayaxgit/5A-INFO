# 5A-INFO
pubblicate le vostre modifiche dei vostri codici qui cosi possiamo vedere le varie modifiche e organizzarci

# Tutorial
## Clonare la repository
Per clonare la repository, cliccate prima sul pulsante verde "< > Code" e copiate l'URL con il tasto dei due quadratini.
Successivamente aprite il Terminale e digitate il seguente codice: winget install git.git
Infine digitate: git clone https://github.com/Ayaxgit/5A-INFO.git e il codice sarà scaricato nella cartella scelta.

## Creare un branch
Un branch è una funzione aggiuntiva del progetto
Per creare un branch andate nella pagina del progetto, poi cliccate Branch e successivamente "New Branch".
Digitate il nome del branch e cliccate su "Create new branch".

## Commit changes
Fare commit significa pubblicare le modifiche svolte con una breve descrizione.
Quando caricate dei file con "Add file" ci sarà un box chiamato "Commit changes".
Qui digitate le vostre modifiche con una breve descrizione.

## Push
Push significa caricare le modifiche dal pc al progetto.
Una volta finite le modifiche, caricate la nuova versione con il commit e la inviate nel branch.

## Pull Request
Pull Request serve per proporre delle modifiche al progetto ma senza modificare la versione originale, ovvero in un branch separato.

Una volta finite le modifiche, aprite un Pull request con una descrizione su cosa avete cambiato e caricate i file,
quando il proprietario della repo controlla il codice e gli va bene clicca su Merge e le modifiche verranno applicate.
